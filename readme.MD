This software is created due to lack of google lens in pc. 
It allows easily read text from image, for example table with data pasted into specification as photo. 

GUI includes: 
  - Button "From clipboard"
  - Button "Choose File" 
  - Text output window

Button "From clipboard" get image from your clipboard. E.g. if you cut some text from image using Snipping tool or Windows Snip(Win+Shift+S) you can click this button to read text from your copied image. 
Button "Choose File" opens choose file dialog window and you need to select the picture you want to read. 
Text output window show readed text from defined photo source. Last readed text is removed with each new reading.
